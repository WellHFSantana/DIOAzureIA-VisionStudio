# DIOAzureIA-VisionStudio
Repository used for Lab2 activities of the Microsoft Azure Fundamentals bootcamp

O apredizado nesse lab foi voltado ao reconhecimento, pela IA, de faces, textos entre outras coisas. Durante esse lab eu pude conhecer as ferramentas e pensar em situações diárias que posso usá-la. 
O conhecimento do arquivo JSON que é criado me fez testar esses resultados em Python, trazendo resultados satisfatórios depois de algumas tentativas.
